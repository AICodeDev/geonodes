# class Vertex

> [main](../index.md) - [nodes](nodes.md) - [nodes menus](nodes_menus.md)

- `verts` [Vertex](Vertex.md)
- `edges` [Edge](Edge.md)
- `faces` [Face](Face.md)
- `corners` [Corner](Corner.md)
